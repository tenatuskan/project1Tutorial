WHAT IS JAVASCRIPT?

JavaScript is one of the three core programming languages of the website after HTML and CSS. It is a scripting language responsible for adding additional behavior and interactivity to your website, meaning you can make the web page alive.

WHAT CAN YOU DO WITH IT?

With JavaScript you can create amazing websites, develop apps & games, and beautiful desktop applications. What makes this programming language unique is its ability to run in your browser directly. It allows to change existing content and modify styles, react to user actions, download and upload files, show messages and many other functions. What we focus on in this tutorial will be JavaScript’s ability to react to keyboard events, that describe a single interaction between a user and a key on the keyboard. Specifically, we will focus on keydown event, that becomes active after a user presses the key on the keyboard.
